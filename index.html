Introduction / Mandate

Need

As a group comprised of UVic students, we noticed the UVic study room reservation website lacks some features. A major inefficiency with the current website is the need to consistently log in to check room availability. That is, 5 minutes is wasted on browsing through the site to check if the desired time slots have opened up. In addition, the current website does not provide adequate directions to the rooms, it only provides the building and room code when reserving a room. The site could also benefit from visual guides of each room so students can find a more suitable study environment.

Purpose

Our vision is to enhance the study room booking experience for students, staff members, and any potential users of the McPherson Library. With this site, we hope to mitigate frustration and increase efficiency while providing a better booking experience. This site introduces new features such as notifications, visual aids, administrator functionality, and interactive maps.

Approach

The Study Room Site allows user to monitor the status of reserved rooms in real time, which allows users to bypass the hassle of constantly checking the UVic library study room bookings. This site prompts users to enter a room number and start/end times, then proceeds to monitor the room's availability. The user can also set a timer on the length of monitoring. If the room is already being monitored then the user will be added to queue and suggested to monitor another room. When the monitored room is available, the site will automatically book the room. Additionally, the site will offer a gallery for each study room and a interactive map to guide users to their destination.
